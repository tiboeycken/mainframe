Modern Web Dashboard for z/OS System Monitoring & COBOL Development

What it is

A comprehensive dashboard system with two components: (1) A USS-based terminal dashboard that displays system info, JES jobs, and datasets at login, and (2) A modern web dashboard running locally that connects via Zowe CLI. The web version includes an integrated COBOL editor with compile/run functionality, real-time job tracking, and spool file viewing.

Why it's AWESOME

  • Dual-mode operation: Terminal dashboard on mainframe + web dashboard locally
  • Integrated COBOL IDE: Edit, upload, compile & run COBOL directly from browser
  • Real-time monitoring: System info, JES jobs, datasets with live updates
  • Production-ready: Clean UI, error handling, session persistence
  • Modern stack: Python, Streamlit, Zowe CLI, JCL integration

Key Features

System Summary JES Job Monitoring Dataset Browser COBOL Editor JCL Generation Job Status Tracking Spool File Viewer Auto-refresh
Built with Python, Streamlit, Zowe CLI, JCL, COBOL, and USS. Source code and instructions available in submission package.
